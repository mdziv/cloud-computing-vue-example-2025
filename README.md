# 2025-cloud-computing-vue-example

Show and browse all the players in one discord server. Frontend for the [spring-boot-cache project](https://github.com/pkresoja/cloud-spring-boot-example-2025).

## Libraries used

- `NuxtUI` - Open-source UI library for Vue or Nuxt
- `TypeScript` - Programming language that adds types to JavaScript

## `.env` Example

```sh
VITE_APP_API_BASE=<URL of server running spring-boot-cache application>/api
```
or if [spring-boot-cache application](https://github.com/pkresoja/cloud-spring-boot-example-2025) is running and set up correctly:

```sh
VITE_APP_API_BASE=/api
```

## Project Setup

```sh
bun install
bun run build
```
